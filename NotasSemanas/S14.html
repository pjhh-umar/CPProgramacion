Matrices

Introducción

Una matriz $M$ de tamaño $m \times n$ es un conjunto finito y homogeneo de datos, es decir, del mismo tipo de dato que se almacenan bajo un mismo nombre, y se diferencian por la posición que tiene cada elemento dentro del arreglo el cual tiene forma rectangular o cuadrada. Una matriz de tamaño $m \times n$ tiene $m$ filas y $n$ columnas.

Para cada $0\leq i \leq n-1 $ y $0\leq j \leq m-1$, el elemento $M(i,j)$ hace referencia a el elemento de la matriz que se encuentra en la fila $i-1$ y columna $j-1$.

Observación

Un matriz de tamaño $3\times 5$ se representa gráficamente como:

$M(0,0)$ $M(0,1)$ $M(0,2)$ $M(0,3)$ $M(0,4)$
$M(1,0)$ $M(1,1)$ $M(1,2)$ $M(1,3)$ $M(1,4)$
$M(2,0)$ $M(2,1)$ $M(2,2)$ $M(2,3)$ $M(2,4)$
